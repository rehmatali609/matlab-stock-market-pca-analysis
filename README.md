# Stock Market Analysis Using Numerical Methods

## CE 342 — Computational Methods and Techniques (Spring 2026)
### Course End Project (CEP)
GIK Institute of Engineering Sciences & Technology

---

## Project Overview

This project applies core numerical methods from the CE 342 syllabus to stock market analysis using MATLAB.

The implementation demonstrates how mathematical and computational techniques can model, analyze, and forecast financial behavior using synthetic stock market datasets.

## Topics Covered

| Part | Numerical Method | Stock Market Application |
|------|------------------|---------------------------|
| 1 | Floating-Point Errors | Error in daily return computation |
| 2 | Root Finding | Break-even stock price (NPV = 0) |
| 3 | Interpolation | Estimating missing stock prices |
| 4 | Curve Fitting | Trend analysis & forecasting |
| 5 | Differentiation & Integration | Momentum & total return |
| 6 | Linear Systems | Portfolio optimization |
| 7 | Eigenvalues & SVD | PCA on stock portfolio |
| 8 | ODE Methods | Geometric Brownian Motion |

## How to Run

1. Open MATLAB
2. Set Current Folder to project directory
3. Run:

```matlab
main
```

4. All 8 project parts execute automatically.
5. Generated plots are saved inside the `plots/` folder.
